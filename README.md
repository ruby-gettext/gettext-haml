# gettext-haml

## Install

```bash
gem install gettext-haml
```

## Usage

If you use `rxgettext`, require `gettext-haml` by `--require`:

```bash
rxgettext --require gettext-haml ...
```

If you use gettext as library, require `gettext-haml` in your Ruby script:

```ruby
require "gettext-haml"
```
